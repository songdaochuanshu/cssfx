# CSSFX

Beautifully simple click-to-copy CSS effects.

**Tech Stack:** React 19 + Vite + TypeScript

## Features

- 30+ CSS effects across 4 categories (Button, Input, Loader, Text)
- Click preview to copy CSS code
- Live code preview with syntax highlighting
- Category filtering and search
- Dark theme, responsive design

## Development

```bash
npm install
npm run dev
```

## Deployment

Auto-deploys to Cloudflare Pages on push to `main`.
